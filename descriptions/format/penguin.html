Missile optimization software made under the guidance of Technical Program Manager at the U.S. Navy.


missile-booster-code

Features

While a booster stage makes a missile easier to launch, it decreases performance. Given some parameters, this program optimizes the ideal stage size for a missile with and without the booster stage. The program then shows graph that compares the performance of both optimized missile over their flight.
